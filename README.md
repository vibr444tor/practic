# GameStore - Магазин видеоигр

Полнофункциональный веб-сайт для просмотра и фильтрации видеоигр с бэкендом на Node.js/Express и фронтендом на HTML/CSS/JavaScript.

## Функциональность

- 📋 Просмотр списка игр с карточками
- 🔍 Поиск игр по названию
- 🎮 Фильтрация по жанрам (Action, RPG, Strategy, Simulation, Adventure)
- 🖥️ Фильтрация по платформам (PC, PlayStation, Xbox, Nintendo)
- 📊 Сортировка по цене, рейтингу и названию
- 📱 Модальное окно с детальной информацией об игре
- ✅ Валидация и обработка ошибок
- 📱 Адаптивный дизайн

## Установка и запуск

### Требования

- Node.js (версия 14 или выше)
- npm (входит в состав Node.js)

### Бэкенд

1. Перейдите в папку backend:

cd backends
2. Установите зависимости:

npm install


Сервер запустится на порту 3000: http://localhost:3000

Для разработки с автоматической перезагрузкой:

npm run dev

### Фронтенд
Откройте файл frontend/index.html в браузере
